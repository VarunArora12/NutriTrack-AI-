# 🥗 NutriTrack AI

A full-stack nutrition tracking web application that helps users log meals, estimate nutrition, and monitor daily calorie and macronutrient intake.

---

## 🚀 Features

- 🔐 Secure user authentication
- 🍽️ Meal logging
- 🧠 Nutrition estimation
- 📊 Daily nutrition dashboard
- 📈 Analytics
- 👤 User profile management
- 🔍 Food search
- 📱 Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Flask
- SQLite

### Deployment
- Vercel
